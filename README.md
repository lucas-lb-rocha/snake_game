# Snake game

Implementação do famoso jogo da cobrinha em Python3.8

### 📋 Pré-requisitos

- Python 3.8+;
- pygame 2.4+

### 🚀  Execução

```
python3.8 game.py
```

### Jogando

Utilize as teclas A, W, S e D para se movimentar pela tela.
